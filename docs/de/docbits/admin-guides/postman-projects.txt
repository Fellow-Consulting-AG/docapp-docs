## Beispiel für die "POST"-Methode

Die POST-Methode wird normalerweise verwendet, um Benutzer oder Organisationen zu erstellen. Diese Methode fügt Informationen in die Datenbank ein.

### Benutzer erstellen

**1.** Wählen Sie die POST-Methode aus.<br>
**2.** Autorisieren Sie sich wie oben beschrieben [Autorisierung](/docbits/admin-guides/postman-projects/#autorisation).<br>
**3.** Öffnen Sie <a href="https://api.polydocs.io">api.polydocs.io</a> und fügen Sie den Pfad der Funktion hinter der Polydocs-URL hinzu. In diesem Fall:

![Picture](/_images/docbits/admin_guides_post_api.png){ loading=lazy }



    https://api.polydocs.io/users/create


**4.** Fügen Sie diesen Link nun in das Textfeld neben der POST-Methode in Postman ein.<br>
**5.** Wählen Sie im HTTP-Request den `Body`-Tab aus und geben Sie für jede Anmeldeinformation, die einen roten Stern neben ihrem Namen hat, die Schlüssel und die Werte ein.

Wenn Sie fertig sind, sollte es so aussehen:

![Picture](/_images/docbits/admin_guide_post_body.png){ loading=lazy }

Wenn Sie ein Admin-Konto erstellen möchten, setzen Sie den `is_admin`-Wert auf **true**.

Klicken Sie abschließend auf `Senden` und Sie können alle Anmeldeinformationen, die Sie in der Antwort festgelegt haben, unten sehen. Dies bedeutet, dass der Benutzer erstellt wurde.

![Picture](/_images/docbits/admin_guides_post_response.png){ loading=lazy }

### Dokument hochladen

Sie können die POST-Methode auch verwenden, um ein Dokument in DocBits (Doc²) hochzuladen.

**1.** Wählen Sie die POST-Methode aus.<br>
**2.** Autorisieren Sie sich wie oben beschrieben [Autorisierung](/docbits/admin-guides/postman-projects/#autorisation).<br>
**3.** Öffnen Sie <a href="https://api.polydocs.io">api.polydocs.io</a> und fügen Sie den Pfad der Funktion hinter der Polydocs-URL hinzu. In diesem Fall:

![Picture](/_images/docbits/admin_guides_post_document-process.png){ loading=lazy }



    https://api.polydocs.io/document/process


**4.** Fügen Sie diesen Link nun in das Textfeld neben der POST-Methode in Postman ein.<br>
**5.** Wählen Sie im HTTP-Request den `Body`-Tab aus und wählen Sie `form-data`.
![Picture](/_images/docbits/admin_guides_post_document-process_body-form-data-file.png){ loading=lazy }
![Picture](/_images/docbits/admin_guides_post_document-process_key-value-send.png){ loading=lazy }
**6.** Geben Sie **file** in das `KEY`-Feld ein, wo Sie das versteckte Dropdown-Menü "File" finden. Wählen Sie `File` und wechseln Sie zum `VALUE`-Feld, wo Sie Ihre Datei auswählen können, indem Sie auf `Dateien auswählen` klicken.
**7.** Wenn Sie auf `Senden` klicken, sollten Sie in der Antwort "success": true sehen.

Es sollte so aussehen:

![Picture](/_images/docbits/admin_guides_upload_body.png){ loading=lazy }
